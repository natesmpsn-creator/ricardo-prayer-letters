# Ricardo Portillo's Prayer Letters

A collection of monthly prayer letters documenting ministry work and updates from Nicaragua.

## About

This site hosts prayer letters from Ricardo Portillo's ministry in Nicaragua. Letters are compiled monthly from his updates and shared with prayer partners.

## View Letters

Visit the site: [natesimpson.github.io/ricardo-prayer-letters](https://natesimpson.github.io/ricardo-prayer-letters)

## How This Works

- **Monthly Updates**: Ricardo sends photos, messages, and updates via Discord
- **Auto-Compilation**: Letters are compiled on the first Tuesday of each month
- **Live Site**: Updates automatically appear on this GitHub Pages site
- **Archive**: All past letters are preserved for prayer history

## Current Letter

The latest prayer letter is always available on the homepage.
